# Indonesia Traffic Accident Analysis (1992–2024)

## Project Overview
This project analyzes 33 years of Indonesia traffic accident data (1992–2024) 
sourced from BPS (Badan Pusat Statistik) — Indonesia's official statistics agency.

## Tools Used
- Excel — data cleaning, calculated columns, dashboard
- SQL (SQLite) — data analysis and querying

## Dataset
- Source: BPS Indonesia (bps.go.id)
- Years: 1992–2024
- Fields: Number of accidents, killed, serious injury, slight injury, material losses

## Key Insights
- Death rate dropped from ~50% (1992) to ~17% (2024) due to stricter traffic laws
- 2020 showed a significant drop in accidents due to COVID-19 lockdowns
- Slight injuries are rising while fatalities remain stable — better vehicle safety
- Record accidents in 2022–2023 following post-COVID traffic surge
- Rural areas need more traffic police and stricter protective gear enforcement
